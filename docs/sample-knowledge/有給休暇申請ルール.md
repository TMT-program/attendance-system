# 有給休暇申請ルール

## 付与日数
有給休暇は入社6ヶ月後に10日、その後は勤続年数に応じて最大20日まで付与されます。

## 申請方法
有給休暇を取得する場合は、3営業日前までに上長へ口頭で連絡し、勤怠管理システムから申請を行ってください。

## 申請手順
1. 勤怠管理システムにログインする
2. 「勤務報告」メニューを開く
3. 該当日の作業区分で「有給休暇」を選択する
4. 備考欄に取得理由を簡潔に記入する
5. 「提出」ボタンを押して申請を完了する

## 承認フロー
申請後、管理者が内容を確認して承認または差し戻しを行います。
承認されると勤務報告のステータスが「承認済」に変わります。

## 注意事項
- 当日申請は原則として認められません。緊急の場合は必ず上長へ電話連絡をしてください。
- 年度末（3月末）に残日数は消滅します。計画的に取得してください。
- 半日有給（午前・午後）の取得も可能です。その場合は備考欄に「午前半休」または「午後半休」と記入してください。
