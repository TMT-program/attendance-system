# 残業申請ルール

## 基本方針
残業は事前申請が原則です。上長の許可なく残業を行った場合、残業代が支払われないことがあります。

## 申請タイミング
残業が見込まれる場合は、当日の18時までに上長へ申請してください。
やむを得ず事前申請ができなかった場合は、翌営業日中に事後申請を行ってください。

## 申請手順
1. 勤怠管理システムにログインする
2. 「勤務報告」メニューを開く
3. 該当日の作業区分で「残業」を選択する
4. 備考欄に残業理由と予定終了時刻を記入する
5. 「提出」ボタンを押して申請を完了する

## 残業時間の上限
法令に基づき、残業時間は月45時間・年360時間を上限とします。
上限に近づいた場合は上長から通知が行われます。

## 深夜残業について
22時以降の勤務は深夜残業として扱われ、通常の1.25倍の割増賃金が適用されます。
深夜残業が見込まれる場合は、必ず事前に上長へ報告してください。

## 注意事項
- 残業申請がない場合、退勤打刻は通常退勤として処理されます。
- 月の残業集計は毎月末日に確定します。
